General Markup

The general template structure is the same throughout the template.

  1. Choose a main color (inside css/colors folder) and include it regularly AFTER the main "style.css" file

  2. If you are using the light version, include the css file "base-light.css" regularly AFTER the main "style.css" file.

  3. Each section must have a ".background1" or ".background2" class.

  4. Remove the rest of the style switcher code before production, HTML and CSS files.

Sections can be reorganized how you need, remember to change the background color.

Navigation

The navigation is very easy to customize, a link must have an "href" and "data-id" matching a section with the same ID on the page. As well as the class ".scroll-link"

A dropdown in structured simply as well:

Animations

The animations on the page are easy to reuse anywhere by simply applying the animation you want as a class (.rightReveal for example) on the element you want to animate. It uses scrollReveal.js.

/* ---- animations ---- */
if (typeof sr == 'undefined') {
    window.sr = ScrollReveal({
        duration: 1500,
        delay: 50
    });
}
Royal_Preloader.config({
    onComplete: function () {
        triggerReveals();
    }
});
function triggerReveals() {
    sr.reveal('.bottomReveal', {
        origin: 'bottom'
    }).reveal('.leftReveal', {
        origin: 'left'
    }).reveal('.rightReveal', {
        origin: 'right'
    }).reveal('.topReveal', {
        origin: 'top'
    });

    sr.reveal('.rotateBottomReveal', {
        origin: 'bottom',
        rotate: { x: 90 }
    }).reveal('.rotateLeftReveal', {
        origin: 'left',
        rotate: { x: 90 }
    }).reveal('.rotateRightReveal', {
        origin: 'right',
        rotate: { x: 90 }
    }).reveal('.rotateTopReveal', {
        origin: 'top',
        rotate: { x: 90 }
    })

    sr.reveal('.scaleReveal', {
        origin: 'top',
        scale: 0.6
    });
}

Magnific Popup Lightbox

This is the js activating our image, video, and gallery zoom. Simply add the corresponding class to your a tag and the link to your image or video as the href. It works the same way for videos, maps, and galleries as well.

/* ---- our work gallery ---- */
$('#work').magnificPopup({
    delegate: 'a.zoom',
    type: 'image',
    fixedContentPos: false,
    removalDelay: 300,
    mainClass: 'mfp-fade',
    gallery: {
        enabled: true,
        preload: [0,2]
    }
});

/* ---- popup image ---- */
$('.popup-img').magnificPopup({
    type: 'image',
    removalDelay: 300,
    mainClass: 'mfp-fade'
});

/* ---- popup video ---- */
$(document).ready(function() {
    $('.popup-youtube, .popup-vimeo, .popup-gmaps').magnificPopup({
        disableOn: 700,
        type: 'iframe',
        mainClass: 'mfp-fade',
        removalDelay: 160,
        preloader: false,
        fixedContentPos: false
    });
});

Parallax.js code

To easily add a parallax effect behind an element, add data-parallax="scroll" to the element you want to use, and specify an image with data-image-src="/path/to/image.jpg".

<div class="parallax-window" data-parallax="scroll" data-image-src="/path/to/image.jpg"></div>
